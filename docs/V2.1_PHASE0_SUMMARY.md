### OpenAI Service
- ✅ 支持模型：GPT-4o-mini, GPT-4o
- ✅ chat() - 文本生成
- ✅ chatJson() - 结构化JSON输出
- ✅ 自动JSON提取
- ✅ 错误处理和日志

### 三层路由策略

| Tier | 主模型 | 备用模型 | 用例 | 预估成本 |
|------|--------|----------|------|----------|
| **Fast** | Claude Haiku 4.5 | GPT-4o-mini | Agent A/B数据结构化、执行Agent | $15-30/月 |
| **Balanced** | Claude Sonnet 4.6 | GPT-4o | Agent C因子分析、日报生成 | $40-80/月 |
| **Powerful** | Claude Opus 4.6 | GPT-4o | Agent D综合决策、Impact Score | $80-200/月 |

### 路由逻辑
1. 尝试主模型（Anthropic优先）
2. 失败则切换备用模型（OpenAI）
3. 全部失败则抛出异常
4. 自动记录usage（inputTokens/outputTokens）

### 便捷方法
- `fastChat()` / `fastChatJson<T>()`
- `balancedChat()` / `balancedChatJson<T>()`
- `powerfulChat()` / `powerfulChatJson<T>()`

## 📡 WebSocket Gateway

### DeliveryGateway特性
- ✅ Socket.io集成
- ✅ CORS配置
- ✅ Tenant房间隔离（`tenant:{tenantId}`）
- ✅ 连接/断开生命周期管理
- ✅ Redis Pub/Sub集成（多实例支持）
- ✅ 认证握手（tenantId）

### 事件类型
- `order:new` - 新订单（触发声音+倒计时）
- `order:updated` - 订单状态更新
- `order:cancelled` - 订单取消
- `platform:status` - 平台连接状态变化

### 客户端连接
```typescript

// 订阅订单事件

  // 播放声音、启动倒计时、更新Kanban

```
